Infrence Problems

Three types of tasks:

  • Marginal probabilities. Compute marginals of variables: \[p(\mathbf{x}_i)=\sum_{\mathbf{x}': x_i'=x_i}p(\mathbf{x}'\mid \mathbf{\theta}).\] (Or conditional probabilities: posterior distribution)

  • Partition function. Compute the normalizing constant: \[Z(\mathbf{\theta})=\sum_{\mathbf{x}}\prod_{C\in\mathcal{C}}\psi_{C}(\mathbf{x}_C)\]

  • Maximum A Posteriori (MAP) Inference. Compute the variable configuration with the hightest probability: \[\hat{\mathbf{x}}=\arg \max_{\mathbf{x}}p(\mathbf{x}\mid \mathbf{\theta}).\]

Compute Marginal Probabilities

  • Let \(\mathbf{X}=\{X_1,\ldots,X_d\}\) denote the set of variable nodes, each of which can range from \(1\) to \(k\); \(F=\{\psi_{\alpha_1},\ldots, \psi_{\alpha_s}\}\) denote the set of factor nodes, where \(\alpha_1, \ldots, \alpha_s \subset \{1,\ldots, d\}\).

  • Compute the marginal probability naively, we do: \[p(\mathbf{X_1}=\mathbf{x}_1)=\frac{1}{Z}\sum_{x_2,\ldots, x_d}\prod_{i=1}^s \psi_{\alpha_i}(\mathbf{x}_{\alpha_i})\]

  • Computational complexity: \(\mathcal{O}(k^{n-1})\).

  • Instead, we choose an ordering \(\mathcal{I}\) among \(\{X_1,\ldots,X_d\}\) and utilize the factorization of \(p(\mathbf{x})\). (Variable Elimination)
